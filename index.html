Guias realizadas

Guia 1

Esta guía introduce los fundamentos del desarrollo web, cubriendo la estructura básica de un documento HTML y la aplicación de estilos mediante CSS. Se practica el uso de selectores básicos, propiedades de texto y fuentes, y las tres formas de incluir CSS en un proyecto: estilos embebidos, internos y externos. También se incluye el uso de Google Fonts y la creación de tablas y layouts sencillos.

Ejemplo 2 Paises Serie

Guia 2

En esta guía se profundiza en el uso de selectores CSS avanzados como descendientes, hijos, adyacentes y de atributos, así como pseudo-clases y pseudo-elementos. También se explora el modelo de caja (margin, border, padding, content) y se realizan ejercicios prácticos como la creación de un tablero de damas usando nth-child() y la maquetación de una página con elementos posicionados y estilizados.

Pelicula Tablero Biografía

Guia 3

Esta guía se centra en el control del layout mediante propiedades de posicionamiento CSS: estático, relativo, absoluto, fijo y flotante. Se practica con display, visibility, overflow y z-index, y se construyen menús horizontales multinivel y diseños fijos y fluidos. También se introduce el uso de flexbox para crear estructuras responsivas.

Peliculas Viajes Cartas Menu

Guia 4

En esta guía se aborda el diseño web adaptable mediante el uso de: Metaetiqueta viewport, Media queries para diferentes breakpoints, Estrategias como Mobile First y Desktop First, Unidades relativas y estructuras fluidas, Imágenes y tablas responsivas. Se incluye la transformación de un sitio fijo a uno totalmente responsivo para móviles, tablets y escritorios.

Index Media Queries
